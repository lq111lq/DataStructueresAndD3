二叉树（binary tree）
二叉树是一颗树，其中每个节点都本能有多于两个的儿子。
二叉查找树（binary search tree）
二叉查找树是一颗二叉树，并且树中的任意节点X，X的左孩子的值小于X的值，X的右孩子的值大于X的值。
平衡二叉树（binary search tree）
平衡二叉树带有平衡条件的二叉查找树

通过代码：
var avlTree = new avlTree()
avlTree.insert([500,200,150,600,650,250,550])
生成一个平衡二叉树。
